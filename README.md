\# 🎯 Project Bullseye



Een intelligent dartsysteem met AI-scoreherkenning, realtime statistieken en een slimme Dartbot.



\## 🚀 Features



\* Webcam scoredetectie

\* Handmatige score-invoer

\* Slimme Dartbot

\* Realtime statistieken

\* Toernooibeheer

\* Docker ondersteuning



\---



\## 🏗️ Tech Stack



| Onderdeel       | Technologie       |

| --------------- | ----------------- |

| Frontend        | Angular           |

| Backend         | ASP.NET Core (C#) |

| AI Service      | Python            |

| Database        | PostgreSQL        |

| Containerisatie | Docker            |



\---



\## 📦 Architectuur



```text

Angular Frontend

&#x20;       │

ASP.NET Core API

&#x20;  │          │

PostgreSQL   Python AI Service

```



\---



\## 🐳 Run locally



```bash

docker compose up --build
```
